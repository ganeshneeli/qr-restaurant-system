# 🍽️ Premium Restaurant Management System

A full-stack solution for modern restaurants, featuring a real-time ordering system, administrative dashboard, and seamless customer experience.

## 🌟 Overview

This project is a comprehensive restaurant management system divided into two main components:
1.  **Frontend:** A high-end React application for customers and restaurant staff.
2.  **Backend:** A robust Node.js API that manages data, authentication, and real-time communications.

## 🚀 Quick Start (Local Development)

To run the entire system locally, follow these steps:

### 1. Backend Setup
```bash
cd backend
npm install
# Configure your .env (see backend/README.md)
npm run dev
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

The application will typically be accessible at:
- **Frontend:** [https://qr-restaurant-system-2.onrender.com](https://qr-restaurant-system-2.onrender.com)
- **Backend API:** [https://qr-restaurant-system-1.onrender.com](https://qr-restaurant-system-1.onrender.com)
